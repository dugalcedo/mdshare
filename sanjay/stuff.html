JavaScript Quick Reference

Methods, Properties & DOM Manipulation

Instance vs Static Methods

Instance Methods

Called on: Individual instances/values of a type

const num = 42.789; num.toFixed(2); // Called on the instance "num"

Static Methods

Called on: The type/constructor itself (not on instances)

Number.isInteger(42); // Called on Number constructor

Number Methods & Properties

Instance number.toFixed(digits)
Formats a number to a fixed number of decimal places, returns a string.
const price = 19.567; console.log(price.toFixed(2)); // "19.57" console.log(price.toFixed(0)); // "20"
Output: "19.57", "20"
Instance number.toString(radix)
Converts a number to a string. Optional radix parameter for different bases (2-36).
const num = 255; console.log(num.toString()); // "255" console.log(num.toString(16)); // "ff" (hexadecimal) console.log(num.toString(2)); // "11111111" (binary)
Output: "255", "ff", "11111111"
Static Number.isFinite(value)
Checks if a value is a finite number (not Infinity, -Infinity, or NaN).
console.log(Number.isFinite(100)); // true console.log(Number.isFinite(Infinity)); // false console.log(Number.isFinite("100")); // false (string!)
Output: true, false, false
Static Number.isInteger(value)
Checks if a value is an integer.
console.log(Number.isInteger(42)); // true console.log(Number.isInteger(42.5)); // false console.log(Number.isInteger("42")); // false (string!)
Output: true, false, false

Boolean Methods

Instance boolean.toString()
Converts a boolean to a string representation.
const isActive = true; console.log(isActive.toString()); // "true" console.log(false.toString()); // "false"
Output: "true", "false"

String Methods & Properties

Property string.length
Returns the number of characters in a string.
const greeting = "Hello World"; console.log(greeting.length); // 11
Output: 11
Instance string.toUpperCase()
Converts all characters to uppercase.
const text = "hello world"; console.log(text.toUpperCase()); // "HELLO WORLD"
Output: "HELLO WORLD"
Instance string.toLowerCase()
Converts all characters to lowercase.
const text = "HELLO WORLD"; console.log(text.toLowerCase()); // "hello world"
Output: "hello world"
Instance string.localeCompare(compareString)
Compares two strings in the current locale. Returns negative if before, positive if after, 0 if equal.
console.log("a".localeCompare("b")); // -1 (a comes before b) console.log("b".localeCompare("a")); // 1 (b comes after a) console.log("hello".localeCompare("hello")); // 0 (equal)
Output: -1, 1, 0
Instance string.split(separator)
Splits a string into an array based on a separator.
const csv = "apple,banana,orange"; console.log(csv.split(",")); // ["apple", "banana", "orange"] const sentence = "Hello World"; console.log(sentence.split(" ")); // ["Hello", "World"]
Output: ["apple", "banana", "orange"], ["Hello", "World"]
Instance string.trim()
Removes whitespace from both ends of a string.
const messy = " hello world "; console.log(messy.trim()); // "hello world" // Also: trimStart() and trimEnd() console.log(messy.trimStart()); // "hello world "
Output: "hello world"
Instance string.includes(searchString)
Checks if a string contains another string. Returns true/false.
const text = "Hello World"; console.log(text.includes("World")); // true console.log(text.includes("world")); // false (case-sensitive) console.log(text.includes("xyz")); // false
Output: true, false, false
Instance string.startsWith(searchString)
Checks if a string starts with another string.
const url = "https://example.com"; console.log(url.startsWith("https")); // true console.log(url.startsWith("http")); // true console.log(url.startsWith("ftp")); // false
Output: true, true, false
Instance string.endsWith(searchString)
Checks if a string ends with another string.
const filename = "document.pdf"; console.log(filename.endsWith(".pdf")); // true console.log(filename.endsWith(".txt")); // false
Output: true, false
Instance string.slice(start, end)
Extracts a section of a string and returns it as a new string.
const text = "Hello World"; console.log(text.slice(0, 5)); // "Hello" console.log(text.slice(6)); // "World" console.log(text.slice(-5)); // "World" (from end)
Output: "Hello", "World", "World"
Instance string.substring(start, end)
Similar to slice, but doesn't support negative indices.
const text = "Hello World"; console.log(text.substring(0, 5)); // "Hello" console.log(text.substring(6, 11)); // "World"
Output: "Hello", "World"
Instance string.replace(searchValue, replaceValue)
Replaces the first occurrence of a value in a string.
const text = "Hello World World"; console.log(text.replace("World", "JavaScript")); // "Hello JavaScript World" (only first occurrence) // Replace all occurrences with replaceAll console.log(text.replaceAll("World", "JavaScript")); // "Hello JavaScript JavaScript"
Only replaces first match (use replaceAll for all)
Instance string.indexOf(searchValue)
Returns the index of the first occurrence, or -1 if not found.
const text = "Hello World"; console.log(text.indexOf("World")); // 6 console.log(text.indexOf("o")); // 4 (first 'o') console.log(text.indexOf("xyz")); // -1 (not found)
Output: 6, 4, -1
Instance string.charAt(index)
Returns the character at a specified index.
const text = "Hello"; console.log(text.charAt(0)); // "H" console.log(text.charAt(4)); // "o" // Can also use bracket notation console.log(text[0]); // "H"
Output: "H", "o", "H"

Array Methods & Properties

Property array.length
Returns the number of elements in an array.
const fruits = ["apple", "banana", "orange"]; console.log(fruits.length); // 3
Output: 3
Instance array.push(element)
Adds one or more elements to the end of an array. Returns new length. Mutates array.
const numbers = [1, 2, 3]; numbers.push(4); console.log(numbers); // [1, 2, 3, 4] numbers.push(5, 6); console.log(numbers); // [1, 2, 3, 4, 5, 6]
Output: [1, 2, 3, 4], [1, 2, 3, 4, 5, 6]
Instance array.pop()
Removes and returns the last element from an array. Mutates array.
const numbers = [1, 2, 3, 4]; const last = numbers.pop(); console.log(last); // 4 console.log(numbers); // [1, 2, 3]
Output: 4, [1, 2, 3]
Instance array.shift()
Removes and returns the first element from an array. Mutates array.
const numbers = [1, 2, 3, 4]; const first = numbers.shift(); console.log(first); // 1 console.log(numbers); // [2, 3, 4]
Output: 1, [2, 3, 4]
Instance array.unshift(element)
Adds one or more elements to the beginning of an array. Returns new length. Mutates array.
const numbers = [3, 4]; numbers.unshift(1, 2); console.log(numbers); // [1, 2, 3, 4]
Output: [1, 2, 3, 4]
Instance array.splice(start, deleteCount, item1, ...)
Adds/removes elements from an array at a specific position. Mutates array.
const arr = [1, 2, 3, 4, 5]; arr.splice(2, 1); // Remove 1 element at index 2 console.log(arr); // [1, 2, 4, 5] const arr2 = [1, 2, 5]; arr2.splice(2, 0, 3, 4); // Insert 3, 4 at index 2 console.log(arr2); // [1, 2, 3, 4, 5]
Output: [1, 2, 4, 5], [1, 2, 3, 4, 5]
Instance array.reverse()
Reverses the order of elements in an array. Mutates array.
const numbers = [1, 2, 3, 4]; numbers.reverse(); console.log(numbers); // [4, 3, 2, 1]
Output: [4, 3, 2, 1]
Instance array.slice(start, end)
Returns a shallow copy of a portion of an array. Does not mutate original.
const fruits = ["apple", "banana", "orange", "grape"]; const citrus = fruits.slice(1, 3); console.log(citrus); // ["banana", "orange"] console.log(fruits); // ["apple", "banana", "orange", "grape"]
Output: ["banana", "orange"], original unchanged
Static Array.isArray(value)
Checks if a value is an array.
console.log(Array.isArray([1, 2, 3])); // true console.log(Array.isArray("hello")); // false console.log(Array.isArray({a: 1})); // false
Output: true, false, false
Instance array.forEach(callback)
Executes a function for each array element. No return value.
const numbers = [1, 2, 3]; numbers.forEach((num, index) => { console.log(`Index ${index}: ${num}`); }); // Output: Index 0: 1, Index 1: 2, Index 2: 3
Executes function for each element
Instance array.map(callback)
Creates a new array by transforming each element with a function.
const numbers = [1, 2, 3, 4]; const doubled = numbers.map(num => num * 2); console.log(doubled); // [2, 4, 6, 8]
Output: [2, 4, 6, 8]
Instance array.filter(callback)
Creates a new array with elements that pass a test function.
const numbers = [1, 2, 3, 4, 5, 6]; const evens = numbers.filter(num => num % 2 === 0); console.log(evens); // [2, 4, 6]
Output: [2, 4, 6]
Instance array.find(callback)
Returns the first element that passes a test function (or undefined if none found).
const users = [ { id: 1, name: "Alice" }, { id: 2, name: "Bob" } ]; const user = users.find(u => u.id === 2); console.log(user); // { id: 2, name: "Bob" }
Output: { id: 2, name: "Bob" }
Instance array.sort(compareFunction)
Sorts array elements in place. Optional compare function for custom sorting. Mutates array.
const numbers = [3, 1, 4, 1, 5, 9]; numbers.sort((a, b) => a - b); // Ascending console.log(numbers); // [1, 1, 3, 4, 5, 9] // Descending numbers.sort((a, b) => b - a); console.log(numbers); // [9, 5, 4, 3, 1, 1] const words = ["banana", "apple", "cherry"]; words.sort(); console.log(words); // ["apple", "banana", "cherry"]
Output: [1, 1, 3, 4, 5, 9], ["apple", "banana", "cherry"]
Instance array.join(separator)
Joins all array elements into a string with a separator.
const words = ["Hello", "World"]; console.log(words.join(" ")); // "Hello World" console.log(words.join("-")); // "Hello-World" const numbers = [1, 2, 3]; console.log(numbers.join()); // "1,2,3" (default comma)
Output: "Hello World", "Hello-World", "1,2,3"
Instance array.concat(array2)
Merges two or more arrays into a new array. Does not mutate original.
const arr1 = [1, 2, 3]; const arr2 = [4, 5, 6]; const combined = arr1.concat(arr2); console.log(combined); // [1, 2, 3, 4, 5, 6] // Can concat multiple const arr3 = [7, 8]; const result = arr1.concat(arr2, arr3); console.log(result); // [1, 2, 3, 4, 5, 6, 7, 8]
Output: [1, 2, 3, 4, 5, 6]
Instance array.includes(element)
Checks if an array contains a specific element. Returns true/false.
const fruits = ["apple", "banana", "orange"]; console.log(fruits.includes("banana")); // true console.log(fruits.includes("grape")); // false const numbers = [1, 2, 3, 4, 5]; console.log(numbers.includes(3)); // true
Output: true, false, true
Instance array.indexOf(element)
Returns the first index of an element, or -1 if not found.
const fruits = ["apple", "banana", "orange", "banana"]; console.log(fruits.indexOf("banana")); // 1 (first occurrence) console.log(fruits.indexOf("grape")); // -1 (not found)
Output: 1, -1
Instance array.every(callback)
Tests if ALL elements pass a test function. Returns true/false.
const numbers = [2, 4, 6, 8]; const allEven = numbers.every(num => num % 2 === 0); console.log(allEven); // true const mixed = [2, 4, 5, 8]; const allEven2 = mixed.every(num => num % 2 === 0); console.log(allEven2); // false
Output: true, false
Instance array.some(callback)
Tests if ANY element passes a test function. Returns true/false.
const numbers = [1, 3, 5, 8]; const hasEven = numbers.some(num => num % 2 === 0); console.log(hasEven); // true (8 is even) const allOdd = [1, 3, 5, 7]; const hasEven2 = allOdd.some(num => num % 2 === 0); console.log(hasEven2); // false
Output: true, false
Instance array.reduce(callback, initialValue)
Reduces an array to a single value by applying a function to each element.
const numbers = [1, 2, 3, 4, 5]; const sum = numbers.reduce((total, num) => total + num, 0); console.log(sum); // 15 // Find maximum const max = numbers.reduce((max, num) => num > max ? num : max); console.log(max); // 5 // Build object const fruits = ["apple", "banana", "apple"]; const count = fruits.reduce((acc, fruit) => { acc[fruit] = (acc[fruit] || 0) + 1; return acc; }, {}); console.log(count); // { apple: 2, banana: 1 }
Powerful method for transforming arrays
Instance array.flat(depth)
Flattens nested arrays by specified depth (default 1).
const nested = [1, [2, 3], [4, [5, 6]]]; console.log(nested.flat()); // [1, 2, 3, 4, [5, 6]] console.log(nested.flat(2)); // [1, 2, 3, 4, 5, 6] console.log(nested.flat(Infinity)); // Fully flatten
Output: [1, 2, 3, 4, [5, 6]], [1, 2, 3, 4, 5, 6]

Object Methods

Static Object.keys(object)
Returns an array of an object's property names (keys).
const person = { name: "Alice", age: 30, city: "NYC" }; const keys = Object.keys(person); console.log(keys); // ["name", "age", "city"] // Useful for looping keys.forEach(key => { console.log(`${key}: ${person[key]}`); });
Output: ["name", "age", "city"]
Static Object.values(object)
Returns an array of an object's property values.
const person = { name: "Alice", age: 30, city: "NYC" }; const values = Object.values(person); console.log(values); // ["Alice", 30, "NYC"]
Output: ["Alice", 30, "NYC"]
Static Object.entries(object)
Returns an array of [key, value] pairs.
const person = { name: "Alice", age: 30 }; const entries = Object.entries(person); console.log(entries); // [["name", "Alice"], ["age", 30]] // Great for looping for (const [key, value] of entries) { console.log(`${key}: ${value}`); }
Output: [["name", "Alice"], ["age", 30]]
Static Object.assign(target, source)
Copies properties from source objects to a target object. Mutates target.
const target = { a: 1, b: 2 }; const source = { b: 3, c: 4 }; Object.assign(target, source); console.log(target); // { a: 1, b: 3, c: 4 } // Common: create new object (shallow copy) const original = { x: 1, y: 2 }; const copy = Object.assign({}, original); console.log(copy); // { x: 1, y: 2 }
Merges objects, later sources override

Math Object

Static Math.round(x)
Rounds to the nearest integer.
console.log(Math.round(4.7)); // 5 console.log(Math.round(4.4)); // 4 console.log(Math.round(4.5)); // 5
Output: 5, 4, 5
Static Math.floor(x)
Rounds down to the nearest integer.
console.log(Math.floor(4.9)); // 4 console.log(Math.floor(4.1)); // 4 console.log(Math.floor(-4.1)); // -5
Output: 4, 4, -5
Static Math.ceil(x)
Rounds up to the nearest integer.
console.log(Math.ceil(4.1)); // 5 console.log(Math.ceil(4.9)); // 5 console.log(Math.ceil(-4.9)); // -4
Output: 5, 5, -4
Static Math.max(...values)
Returns the largest of the given numbers.
console.log(Math.max(1, 5, 3, 9, 2)); // 9 // With array, use spread operator const numbers = [1, 5, 3, 9, 2]; console.log(Math.max(...numbers)); // 9
Output: 9
Static Math.min(...values)
Returns the smallest of the given numbers.
console.log(Math.min(1, 5, 3, 9, 2)); // 1 const numbers = [1, 5, 3, 9, 2]; console.log(Math.min(...numbers)); // 1
Output: 1
Static Math.random()
Returns a random number between 0 (inclusive) and 1 (exclusive).
console.log(Math.random()); // e.g., 0.7392... // Random integer between min and max (inclusive) function randomInt(min, max) { return Math.floor(Math.random() * (max - min + 1)) + min; } console.log(randomInt(1, 10)); // Random number 1-10
Output: 0 ≤ x < 1
Static Math.abs(x)
Returns the absolute (positive) value of a number.
console.log(Math.abs(-5)); // 5 console.log(Math.abs(5)); // 5 console.log(Math.abs(-3.7)); // 3.7
Output: 5, 5, 3.7
Static Math.pow(base, exponent)
Returns base raised to the power of exponent.
console.log(Math.pow(2, 3)); // 8 (2³) console.log(Math.pow(5, 2)); // 25 (5²) // Can also use ** operator console.log(2 ** 3); // 8
Output: 8, 25, 8
Static Math.sqrt(x)
Returns the square root of a number.
console.log(Math.sqrt(16)); // 4 console.log(Math.sqrt(25)); // 5 console.log(Math.sqrt(2)); // 1.414...
Output: 4, 5, 1.414...

Console Methods

console.log(...values)
Outputs messages to the browser console.
console.log("Hello"); console.log("Multiple", "values", 123); const user = { name: "Alice", age: 30 }; console.log(user); // Shows object structure // Template literals console.log(`User: ${user.name}`);
Primary debugging tool
console.error(message)
Outputs an error message (styled red in console).
console.error("Something went wrong!"); console.error("Error:", new Error("Failed"));
Displays as error in console
console.table(data)
Displays arrays or objects as a formatted table.
const users = [ { name: "Alice", age: 30 }, { name: "Bob", age: 25 } ]; console.table(users); // Shows nice table in console
Great for viewing structured data

DOM Manipulation

document.getElementById(id)
Returns the element with the specified ID.
const header = document.getElementById("main-header"); console.log(header); // <div id="main-header">...</div>
Returns single element or null
document.querySelector(selector)
Returns the first element matching a CSS selector.
const firstButton = document.querySelector(".btn"); const nav = document.querySelector("#navbar .active"); console.log(firstButton); // First element with class "btn"
Returns first matching element or null
document.querySelectorAll(selector)
Returns all elements matching a CSS selector as a NodeList.
const allButtons = document.querySelectorAll(".btn"); console.log(allButtons.length); // Number of buttons // Convert to array for array methods const buttonArray = Array.from(allButtons); buttonArray.forEach(btn => { console.log(btn.textContent); });
Returns NodeList (array-like, not array)
document.createElement(tagName)
Creates a new HTML element (not yet added to DOM).
const newDiv = document.createElement("div"); newDiv.className = "container"; const newButton = document.createElement("button"); newButton.textContent = "Click me";
Creates new element in memory
Instance element.innerText
Gets or sets the visible text content of an element.
const heading = document.querySelector("h1"); console.log(heading.innerText); // Gets text heading.innerText = "New Title"; // Sets text
Read/write property for text content
Instance element.innerHTML
Gets or sets the HTML content inside an element. Can parse HTML tags.
const container = document.querySelector(".container"); container.innerHTML = "<p>Hello <strong>World</strong></p>"; console.log(container.innerHTML); // Gets HTML string
⚠️ Be careful with user input (XSS risk)
Instance element.textContent
Gets or sets the text content of an element (including hidden text). Safer than innerHTML.
const para = document.querySelector("p"); para.textContent = "Safe text with <tags>"; // Displays literally: "Safe text with <tags>" // Unlike innerText, includes hidden elements console.log(para.textContent);
HTML tags are displayed as text, not parsed
Instance element.classList.add/remove/toggle/contains
Manipulates CSS classes on an element.
const box = document.querySelector(".box"); box.classList.add("active"); // Add class box.classList.remove("inactive"); // Remove class box.classList.toggle("hidden"); // Toggle on/off box.classList.contains("active"); // Check if has class (true/false) // Add multiple box.classList.add("class1", "class2", "class3");
Modern way to manipulate classes
Instance element.style.property
Gets or sets inline CSS styles on an element.
const box = document.querySelector(".box"); box.style.backgroundColor = "blue"; box.style.width = "200px"; box.style.display = "none"; // Note: CSS properties use camelCase // background-color → backgroundColor // font-size → fontSize
Sets inline styles (high specificity)
Instance element.setAttribute(name, value)
Sets the value of an attribute on an element.
const img = document.querySelector("img"); img.setAttribute("src", "photo.jpg"); img.setAttribute("alt", "A beautiful photo"); img.setAttribute("data-id", "123");
Adds or updates attribute
Instance element.getAttribute(name)
Returns the value of a specified attribute.
const link = document.querySelector("a"); const url = link.getAttribute("href"); const target = link.getAttribute("target"); console.log(url); // Gets href value
Returns attribute value or null
Instance element.removeAttribute(name)
Removes an attribute from an element.
const input = document.querySelector("input"); input.removeAttribute("disabled"); // Enable input input.removeAttribute("readonly");
Completely removes the attribute
Instance element.addEventListener(type, listener, options)
Attaches an event handler to an element. Common events: click, input, change, submit, keydown, mouseover, etc.
const button = document.querySelector("button"); button.addEventListener("click", (e) => { console.log("Button clicked!"); console.log(e.target); // The clicked element }); // Multiple event types const input = document.querySelector("input"); input.addEventListener("input", (e) => { console.log("Value changed:", e.target.value); }); // Form submission const form = document.querySelector("form"); form.addEventListener("submit", (e) => { e.preventDefault(); // Stop form from submitting console.log("Form submitted!"); }); // Options: run once then remove button.addEventListener("click", handler, { once: true });
Executes callback when event fires
Instance element.removeEventListener(type, listener)
Removes an event handler from an element. Must use same function reference.
function handleClick() { console.log("Clicked!"); } button.addEventListener("click", handleClick); button.removeEventListener("click", handleClick); // Removed
Function reference must match
Property element.value
Gets or sets the value of form elements.
// Elements with .value property: // <input>, <textarea>, <select> const input = document.querySelector("input"); console.log(input.value); // Get current value input.value = "New text"; // Set value const select = document.querySelector("select"); console.log(select.value); // Selected option value
Only on form controls: input, textarea, select
Instance element.append(content)
Adds content to the end of an element (can be elements or text). Modern method.
const container = document.querySelector(".container"); const newDiv = document.createElement("div"); newDiv.textContent = "New item"; container.append(newDiv); // Add element container.append("Some text"); // Add text container.append(newDiv, "More text"); // Add multiple
Appends to end of element
Instance element.prepend(content)
Adds content to the beginning of an element. Modern method.
const list = document.querySelector("ul"); const newItem = document.createElement("li"); newItem.textContent = "First item"; list.prepend(newItem); // Adds to start of list
Prepends to beginning of element
Instance element.remove()
Removes the element from the DOM. Modern method.
const oldElement = document.querySelector(".old"); oldElement.remove(); // Element is gone from DOM
Simple removal of element
Instance parent.removeChild(child)
Removes a child element. Older method, but still common.
const parent = document.querySelector(".parent"); const child = document.querySelector(".child"); parent.removeChild(child);
Returns the removed child
Instance element.insertAdjacentHTML(position, html)
Inserts HTML at a specified position relative to the element.
const div = document.querySelector("div"); // Positions: "beforebegin", "afterbegin", "beforeend", "afterend" div.insertAdjacentHTML("beforebegin", "<p>Before div</p>"); div.insertAdjacentHTML("afterbegin", "<p>First child</p>"); div.insertAdjacentHTML("beforeend", "<p>Last child</p>"); div.insertAdjacentHTML("afterend", "<p>After div</p>");
4 positions: before/after begin/end
Instance element.insertAdjacentElement(position, element)
Inserts an element at a specified position relative to the element.
const container = document.querySelector(".container"); const newPara = document.createElement("p"); newPara.textContent = "New paragraph"; // Same positions as insertAdjacentHTML container.insertAdjacentElement("afterbegin", newPara); // newPara is now first child of container
Insert element nodes at specific positions
